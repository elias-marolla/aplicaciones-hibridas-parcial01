API RESTful de Nutrición

Gestión de alimentos, categorías y valores nutricionales

¿De qué trata esta API?

A continuación explico de qué trata esta API y cómo funciona. Si querés podés pasar directamente a la parte de los endpoints.

Breve Descripción

Esta API REST permite trabajar con información nutricional de alimentos, categorías y usuarios. Su objetivo es ofrecer datos estructurados (calorías, proteínas, carbohidratos, grasas) para que desarrolladores, nutricionistas o aplicaciones de salud puedan gestionar información nutricional.

¿Qué ofrece la API?

  • Gestión completa de usuarios
  • Catálogo de alimentos con información nutricional detallada
  • Sistema de categorías para organizar alimentos
  • Operaciones CRUD completas para todos los recursos
  • Búsqueda y filtrado de alimentos

Endpoints

👤 Usuarios
GET /api/users Obtener todos los usuarios
POST /api/users
Ver body requerido 
{
  "name": "String (required)", 
  "email": "String (required)", 
  "password": "String (required)"
}
DELETE /api/users/:id Eliminar usuario por ID
🥗 Alimentos
GET /api/alimentos Obtener todos los alimentos
GET /api/alimentos/:id Obtener alimento por ID
POST /api/alimentos
Ver body requerido 
{
  "nombre": "String (required)", 
  "calorias": "Number (required)", 
  "proteinas": "Number (required)",
  "carbohidratos": "Number (required)",
  "grasas": "Number (required)",
  "descripcion": "String (required)"
}
PUT /api/alimentos/:id Todos los campos del POST son editables
DELETE /api/alimentos/:id Eliminar alimento por ID
GET /api/alimentos/search?nombre=Pollo Obtener alimentos por nombre
GET /api/alimentos/filter?minProteinas=20&maxCalorias=300 Filtrar alimentos por Proteinas y Calorias
📂 Categorías
GET /api/categorias Obtener todas las categorías
GET /api/categorias/:id Obtener categoría por ID
POST /api/categorias
Ver body requerido 
{
  "nombre": "String (required)", 
  "descripcion": "String (required)"
}
PUT /api/categorias/:id Todos los campos del POST son editables
DELETE /api/categorias/:id Eliminar categoría por ID
