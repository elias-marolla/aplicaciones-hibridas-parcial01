🥗 API RESTful de Nutrición

Bienvenido a la API para gestionar alimentos, categorías y valores nutricionales (calorías, proteínas, grasas, carbohidratos y vitaminas).

📌 Endpoints principales

  • /api/alimentos → Ver todos los alimentos
  • /api/alimentos/search?nombre=pollo → Buscar alimento por nombre
  • /api/alimentos/filter → Filtrar alimentos
  • /api/categorias → Ver categorías
  • /api/users → Ver usuarios
